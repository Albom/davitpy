fan.plotFan(dateStr, rad, time=[0, 0], interval=60, fileType='fitex', param='velocity', scale=[], channel='a', coords='geo', colors='lasse', gsct=0, pdf=0)[source]¶

plotFan(dateStr,rad,beam,[time],[fileType]):

crate an rti plot for a secified radar and time period

INPUTS:

dateStr: a string containing the target date in yyyymmdd format rad: the 3 letter radar code, e.g. ‘bks’ [time]: the range of times for which the file should be read,

in SIMPLIFIED [hhmm,hhmm] format, eg [29,156], instead of [0029,0156] note that an end time of 2400 will read to the end of the file default = [0,0]

[interval]: the interval between fan plots, in seconds

OUTPUTS:
