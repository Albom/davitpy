PACKAGE: pydarn.io.gridIo FUNCTION: readPygridRec(myFile):

reads pygrid records from sEpoch to eEpoch into a grid structure

INPUTS:

myFile: the pygrid file to read from myGrid: the pydarn.proc.pygridLib.pygrid object to be filled sEpoch: read times >= this time (in epoch) eEpoch: read times < this time (in epoch)
