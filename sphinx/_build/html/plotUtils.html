plotUtils.genCmap(fig, coll, param, scale, pos=[0, 0, 1, 1], colors='lasse', map=0)[source]¶

genCmap(fig,coll,param,scale,pos,[colors]):

generates a colormap and plots a colorbar

INPUTS:

fig: the parent figure coll: the collection of items (e.g. polygons) being mapped

to this colormap
param: the parameter being plotted, valid for ‘velocity’,
‘power’, ‘width’, ‘elevation’, ‘phi0’

scale: the max and min values of the color scale in list form pos: the position of the parent plot, NOT of the COLORBAR [colors]: a string indicating which colorbar to use, valid

inputs are ‘lasse’, ‘aj’ default: ‘lasse’

OUTPUTS:

EXAMPLE:
genCmap(myFig,polyCollection,’velocity’,[-200,200],pos)
